# Audit Matrix

Generate the latest matrix with:

```bash
npm run audit:matrix > docs/audit/audit-matrix.generated.md
```

Manual audit rows should use this schema:

| Area | Journey/API | Role | Environment | Test Type | Expected Result | Actual Result | Severity | Owner | Fix Status |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Baseline | Repo health | developer | local | static | Repo clean and current | TBD | TBD | TBD | Not started |
